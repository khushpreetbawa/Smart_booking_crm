# 📅 Smart Booking CRM

A full-stack appointment booking and CRM platform that helps service-based businesses manage customers, appointments, payments, and support tickets through a modern, responsive dashboard.

## ✨ Features

- 🔐 Secure Authentication & Authorization
- 📅 Appointment Booking & Scheduling
- 👥 Customer Relationship Management (CRM)
- 💳 Razorpay Payment Integration
- 🔔 Real-time Notifications
- 🎫 Support Ticket Management
- 📊 Admin Dashboard & Analytics
- 🤖 AI Chat Assistant
- 👤 User Profile Management
- 📱 Responsive UI

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js
- Socket.io

### Database & Services
- Supabase (PostgreSQL)
- Razorpay
- Nodemailer
- JWT Authentication

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/yourusername/smart-booking-crm.git
cd smart-booking-crm
```

### Install Dependencies

#### Client

```bash
cd client
npm install
npm run dev
```

#### Server

```bash
cd server
npm install
npm run dev
```

## 📌 Environment Variables

Create a `.env` file inside the `server` folder and add the required credentials:

```env
SUPABASE_URL=
SUPABASE_SERVICE_ROLE_KEY=
JWT_SECRET=
RAZORPAY_KEY_ID=
RAZORPAY_KEY_SECRET=
EMAIL_USER=
EMAIL_PASS=
```
